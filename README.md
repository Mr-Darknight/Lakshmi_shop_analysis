# Lakshmi Shop Analysis

A comprehensive data analysis project for Lakshmi Shop that combines Python-based analytics with Power BI visualization for business insights.

## 📊 Project Overview

This project analyzes Lakshmi Shop's operational data including purchase patterns, shop entries, and performance metrics. It provides actionable insights through exploratory data analysis and interactive dashboards.

## ✨ Features

- **Data Analysis**: Pandas-based exploratory data analysis (EDA)
- **Optimization**: Shop optimization analysis with advanced analytics
- **Visualization**: Interactive Power BI dashboard for decision-making
- **Data Processing**: Comprehensive data cleaning and transformation

## 📁 Project Structure

```
Lakshmi_shop_analysis/
├── lakshmi_shop_analysis_pandas.ipynb    # Main EDA notebook with Pandas

├── Purchase_Data.xlsx                    # Purchase transaction data
├── ShopEntry.xlsx                        # Shop entry records
├── LICENSE                               # MIT License

```

## 🔍 Data Sources

- **Purchase_Data.xlsx**: Contains detailed purchase transaction records
- **ShopEntry.xlsx**: Records of shop entries and operational data

## 🚀 Getting Started

### Prerequisites

- Python 3.7+
- Jupyter Notebook
- pandas
- numpy
- matplotlib/seaborn (for visualizations)
- Power BI Desktop (for dashboard)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Mr-Darknight/Lakshmi_shop_analysis.git
cd Lakshmi_shop_analysis
```

2. Install required packages:
```bash
pip install pandas numpy matplotlib seaborn jupyter
```

3. Start Jupyter Notebook:
```bash
jupyter notebook
```

## 📖 Usage

1. **Run Analysis**: Open and run the Jupyter notebooks in this order:
   - `lakshmi_shop_analysis_pandas.ipynb` - Start with basic EDA
   - `ShopOptim.ipynb` - Review optimization analysis

2. **View Dashboard**: Open `ShopDash.pbix` with Power BI Desktop to interact with visualizations

3. **Explore Data**: Check the Excel files for raw data inspection
